Apollo API

Modern

Apollo uses the latest and greatest JavaScript technologies to create a sleek, modern web application. The blazing fast, responsive system is built on a sturdy MERN stack, with a flexible and transparent REST API to help you manage a dependable, ACID compliant noSQL database.

Robust

Apollo's powerful cross-platform API provides the tools for easy and accessible management of the database from any system. All transactions are executed using a replica set for redundancy and high availability, eliminating database errors and corruption.

Scalable

The client-server model allows for distributed systems to syncrhonise their databases for a seamless library experience. Additionally, the server can be configured with MongoDB's native sharding to support larger workloads, so you never have to worry about downtime, wherever you are.

Rich Tagging

Create, edit, and delete tags in seconds, to make finding and organising books that much easier.
